抓 xml/json 工具介紹¶

utils.get_gJX (gzipped JSON and XML)

這部分是 2組4個 工具。功能是將網路上的 json/xml/gzip json/gzip xml 轉成對應的 dict 並回傳。 分別是:

  • get_gJX.HandleGZippedXML(a_url)
  • get_gJX.HandleNonGZippedXML(a_url)
  • get_gJX.HandleGZippedJSON(a_url)
  • get_gJX.HandleNonGZippedJSON(a_url)

幕後的工具寫好放在那邊，不介紹白不介紹。 鏈接： jupyter notebook
